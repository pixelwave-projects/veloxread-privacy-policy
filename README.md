# VeloxRead Privacy Policy

This repository hosts the privacy policy for the VeloxRead app, part of the Pixelwave projects.

The policy is published as a static site via GitHub Pages and is intended to be used as the public Privacy Policy URL for app store submissions and related documentation.

Once GitHub Pages is enabled, the live policy will be available at:

`https://<your-account>.github.io/veloxread-privacy-policy/`
